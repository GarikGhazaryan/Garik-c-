1.The program is designed to store data and retrieve them in a specific order.
  From this storage you can get only the data that was entered first, they can be read and deleted, there is no access to other data.
 
  The program has methods.
  <push> adding eletants,
  <front> getting the first element,
  <size> getting the number of elements,
  <is_empty> check for the presence of elements,
  <pop> getting and removing the first element.

2.Compiling and running the program is done using Makefile commands.
  The commands is.
  <make> compiles the written code into an exe file named Queue.
  <make test> compiles the written code into an exe file and runs it.
  <make clean> removes all creation files when compiling.
